# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — 2026-03-12

### Added
- Initial open-source release
- 7-stage AI content pipeline (Stage 0–6 + 3.5)
- 8 quality gates (G1–G8)
- Google Gemini AI integration with web grounding
- OpenAI API support (chat completions)
- WordPress REST API publish with Rank Math SEO
- CLI: `wcap init`, `wcap start`, `wcap run`, `wcap status`
- Cron scheduler with timezone support
- SQLite database with 12 migration versions
- RSS news feed ingestion
- CSV keyword bulk import
- Content enrichment (hero image, TOC)
- Internal links injection from sitemap
- Daily cost cap and quota management
- Configurable prompts via markdown files
- Docker support (Dockerfile + docker-compose)
- Comprehensive prompt writing guide
